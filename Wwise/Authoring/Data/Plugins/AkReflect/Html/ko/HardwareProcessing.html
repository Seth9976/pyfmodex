Use Hardware Processing

일부 플랫폼에서만 사용할 수 있는 속성입니다. 이 속성은 허음원 처리를 위해 하드웨어 가속을 활용합니다. 이 옵션을 활성화하면 Reflect의 CPU 사용량을 줄일 수 있습니다.

PS5에서 활성화된 경우 다음과 같은 제한이 있습니다: - 최대 허음원 딜레이 시간은 Max Image Source Delay Time 속성을 통해 명시적으로 지정해야 합니다. 이 값을 초과하는 딜레이를 가진 허음원은 렌더링되지 않습니다. - 음속은 RTPC로 사용할 수 없습니다. - Material 필터는 Favor Performance 모드에서만 렌더링됩니다.
