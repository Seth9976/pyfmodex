Output Config

Reflect는 입력과 다른 채널 구성으로 신호를 출력할 수 있는 입출력 분리형 Effect입니다. Output Config를 통해 디자이너는 초기 반사음이 공간화될 채널 구성을 선택할 수 있습니다.

값이 Parent Bus로 설정된 경우, Reflect는 삽입된 버스의 상위 버스 구성을 사용합니다.

출력 구성이 Audio Objects로 설정되면, Reflect는 초기 반사음 각각 하나의 Audio Object를 출력합니다. 이로 인해 많은 수의 Audio Object가 생성될 수 있음을 유의하세요. Output Config는 Audio Objects가 아닌 공간화된 다른 구성으로 설정할 것을 권장합니다.

Default value: Same as main mix
