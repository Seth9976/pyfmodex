Delay Smoothing

Applies a smoothing filter to each reflection time. Increasing the smoothing causes reflection times to update more gradually, which helps limit the Doppler pitch shift caused by movement.

Delay Smoothing is especially useful for reducing artifacts caused by abrupt or irregular position updates from the game engine—such as when the game and audio engine run at different update rates or are not perfectly synchronized.

The normalized value is between 0 and 1 and will not be updated if the reflect length/delay is actively changing. This parameter can be controlled by RTPCs and States; but, updating of the RTPC or State during game play is not recommended. Real-time updates are honored but not guaranteed to be handled without artifacts.

Default value: 0.5
Range: 0 to 1
