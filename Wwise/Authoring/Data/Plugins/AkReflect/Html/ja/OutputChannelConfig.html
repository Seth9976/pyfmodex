Output Config

Reflectは、場外エフェクトのため、入力と異なるチャンネル構成で信号を出力できる。デザイナーはOutput Configを使用し、アーリーリフレクションのスペーシャリゼーションを行うチャンネル構成を選択できる。

この値がParent Busである場合、Reflectは挿入されたバスの親のバス構成を照会し、それを使用する。

なお出力設定がAudio Objectsの場合、Reflectは1つのアーリーリフレクションに対し1つのAudio Objectを出力する。これは相当数のAudio Objectになる可能性がある。Output Configは、Audio Objects以外のスペーシャリゼーション構成に設定することが推奨される。

Default value: Same as main mix
