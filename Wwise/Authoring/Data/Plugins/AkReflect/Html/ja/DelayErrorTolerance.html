Delay Error Tolerance

実効ディレイと目標ディレイの許容される最大誤差を設定する。Pitch Limitによってディレイの変化速度が制限されていると、実効ディレイと目標ディレイで誤差が生じる場合がある。この差が指定した許容値を超えると、システムが実効ディレイと目標ディレイの間に短いクロスフェードを適用し、ディレイを調整する。

許容値は、目標ディレイに対するおおよその割合で示す。例えば許容値が25%で目標ディレイが100ミリ秒の場合、許容される最大誤差は25ミリ秒となる。ただし、このマッピングは厳密に線形ではなく、100%の場合は誤差の大きさにかかわらず、すべて許容されることを意味する。

オーディオコンテンツによっては、クロスフェードがわずかなアーティファクトを引き起こす可能性がある。Delay Error Toleranceを設定する際は、許容できるタイミング誤差とクロスフェードによってアーティファクトが生じる可能性とのバランスを取ることになる。

この値はリフレクションの長さやディレイが変化している最中は、アップデートされない。このパラメータをRTPCやStateでコントロールすることができるが、ゲームプレイ中にRTPCやStateをアップデートすることは、推奨されない。リアルタイムのアップデートは尊重されるが、ノイズなしで扱われるとは限らない。

Default value: 25
Range: 0 to 100
