Pitch Limit

許容される最大Dopplerピッチシフト（セント単位）を設定する。ピッチシフトはディレイ時間の変化により発生するため、ディレイ変化速度に上限を設け、指定したピッチリミット内に収まるようにする。リミットに従うために速度が低下すると、実効ディレイと目標ディレイの間に誤差が生じる場合がある。この誤差はDelay Error Toleranceプロパティで管理できる。

この値はリフレクションの長さやディレイが変化している最中は、アップデートされない。このパラメータをRTPCやStateでコントロールすることができるが、ゲームプレイ中にRTPCやStateをアップデートすることは、推奨されない。リアルタイムのアップデートは尊重されるが、ノイズなしで扱われるとは限らない。

Default value: 300
Range: 0 to 9600
