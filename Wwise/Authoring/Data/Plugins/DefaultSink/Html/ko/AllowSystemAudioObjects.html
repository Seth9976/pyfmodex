Allow System Audio Objects

3D 오디오가 활성화된 상태에서 이 옵션을 선택하면, 엔드포인트가 지원하는 Audio Object 수가 지정된 스레숄드(즉, Minimum System Audio Objects Required 값)을 초과할 경우 Audio Device가 System Audio Object를 엔드포인트로 전송할 수 있도록 허용합니다.

이 옵션을 해제하면 개별 System Audio Object 전송을 비활성화합니다.

이 옵션을 해제하면 Audio Object는 공간 특성을 유지한 채 Main Mix로 믹스됩니다. 이때 공간의 정확도는 Main Mix의 구성에 따라 달라집니다.

Default value: true
