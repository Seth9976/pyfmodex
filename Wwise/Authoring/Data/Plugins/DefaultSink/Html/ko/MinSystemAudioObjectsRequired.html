Minimum System Audio Objects Requirement

사운드 엔진이 System Audio Objects를 생성할때 엔드포인트가 지원해야 하는 최소 System Audio Objects 수를 정의합니다.

이 속성은 시스템의 3D 오디오를 위한 특정 공간을 보장하기 위해 사용할 수 있습니다. 예를 들어, 이 값을 50으로 설정하면 최소 50개의 동시 Audio Objects를 지원하지 않는 플랫폼에서는 System Audio Objects 사용이 비활성화됩니다. 이렇게 하면 시스템이 지원하는 것보다 더 많은 오브젝트를 재생하려고 할 때 일부 오브젝트가 다르게 믹싱되는 상황을 방지할 수 있습니다. 그런 경우에는 오디오가 매번 다르게 들리게 됩니다.

따라서 이 값을 게임이 한 번에 재생할 System Audio Objects 수보다 조금 더 높게 설정하는 것이 좋습니다.

Default value: 32
Range: 0 to 65535
