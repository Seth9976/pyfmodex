Channel Link Percentage

信号内のほかのチャンネルがゲインリダクションに寄与する割合（%）。例えばChannel Link Percentageが60%の場合、出力チャンネルは自身に対して40%のゲインリダクションを適用し、ほかのチャンネルが合算して残りの60%に寄与する（ほかに2チャンネルあればそれぞれ30%、3チャンネルあればそれぞれ20%というようにリダクションを適用する）。

Channel Link Percentageは、Channel Linkが無効の時のみオーディオ入力を変更する。Object-Basedコンフィギュレーションで実行した場合も動作は同じである。

100%に設定すると、すべてのチャンネルに同じゲインリダクションが適用される。

このオプションを0%に設定すると、チャンネル間で情報を共有せず、Effectはチャンネルごとに適用される。

Default value: 0
Range: 0 to 100
