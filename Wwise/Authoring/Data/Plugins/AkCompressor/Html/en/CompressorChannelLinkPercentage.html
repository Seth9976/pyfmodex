Channel Link Percentage

The percentage of the gain reduction contributed by the other channels in the signal. For example, a Channel Link Percentage of 60% means that the output channel applies 40% of the gain reduction on itself and the other channels combined contribute the remaining 60% of the gain reduction (if there are two other channels, each applies a 30% reduction; if there are three other channels, each applies a 20% reduction, and so on).

The channel link percentage only modifies the audio input when Channel Link is disabled. The behavior is the same when running on an Object-Based configuration.

When set to 100%, the same gain reduction is applied to all channels.

If this option is set to 0%, no information is shared between channels, and the Effect is applied on a per-channel basis.

Default value: 0
Range: 0 to 100
