Channel Link Percentage

声道链接比率。定义信号当中由其他声道贡献的增益衰减比率。比如，若 Channel Link Percentage 为 60%，则表示输出声道为自身应用了 40% 的增益衰减，其他声道贡献了剩下 60% 的增益衰减（若其他声道有两个，则其分别应用 30% 的衰减；若其他声道有三个，则其分别应用 20% 的衰减…）。

Channel Link Percentage 只会在禁用 Channel Link 时修改音频输入。在基于对象的配置下运行时，行为是一样的。

若设为 100%，将为所有声道应用相同的增益衰减。

若设为 0%，声道之间将不会共享信息，各声道将单独应用效果器。

Default value: 0
Range: 0 to 100
