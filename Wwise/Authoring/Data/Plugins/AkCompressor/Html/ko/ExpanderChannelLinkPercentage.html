Channel Link Percentage

신호에서 다른 채널들이 기여하는 게인이 감소되는 비율입니다. 예를 들어, Channel Link Percentage가 60%이면 출력 채널은 40%의 게인 감소를 자체적으로 적용하고, 다른 채널들이 나머지 60%의 게인 감소를 합쳐서 적용합니다(다른 채널이 2개라면 각 채널이 30%씩 감소를 적용하고, 3개라면 각 채널이 20%씩 적용하는 식).

Channel Link가 비활성화되어 있을 때만 해당 채널 링크 비율이 오디오 입력에 영향을 줍니다. Object-Based 구성에서 실행될 때도 이는 동일하게 동작합니다.

이를 100%로 설정하면 모든 채널에 동일한 게인 감소가 적용됩니다.

이 옵션을 0%로 설정하면 채널 간 정보가 공유되지 않으며, Effect는 채널별로 개별 적용됩니다.

Default value: 0
Range: 0 to 100
