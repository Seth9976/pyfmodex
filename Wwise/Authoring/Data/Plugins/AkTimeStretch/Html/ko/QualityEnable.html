Stretch Mode

사용할 타임 스트레치 알고리즘 버전을 설정합니다.

다음은 사용 가능한 옵션입니다:

  • Classic: 기존 Time Stretch 알고리즘을 사용하며, Wwise 2021.1 버전 이하에서 사용된 방식입니다.
  • Transient Preserving: 개선된 알고리즘을 사용하는 모드입니다. 이 모드를 사용하면 Quality Level 슬라이더와 함께 알고리즘 성능을 세밀하게 조정할 수 있으며, Stereo Processing 컨트롤을 통해 복잡한 스테레오 믹스를 더 잘 처리할 수 있습니다.

Default value: Classic
