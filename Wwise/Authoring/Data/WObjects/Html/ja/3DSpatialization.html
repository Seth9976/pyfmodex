3D Spatialization

チェックを入れると、ソースのポジショニングを計算して、3D環境を移動する様子がシミュレーションされる。以下のオプションから選択する：

  • None（デフォルト）：サウンドはSpeaker Panning設定に従いパンニングされる。
  • Position：サウンドはエミッターとリスナーの相対位置に従いパンニングされる。
  • Position + Orientation：サウンドはエミッターとリスナーの相対位置に従いパンニングされる。またエミッターとリスナーの相対的な向きに応じて、サウンドのマルチチャンネルコンテンツも回転する。Orientationの影響が出るのは、インプットファイルがマルチチャンネルで、スプレッドが0より大きい場合に限る。

Default value: None
