Playback Speed

음악 오브젝트의 컨텐츠가 재생될 속도를 조정합니다:

  • 값이 1보다 클 경우 보통보다 빠른 속도로 재생됩니다
  • 값이 1보다 적을 경우 보통보다 느린 속도로 재생됩니다
  • 값이 1일 경우 보통의 속도로 재생됩니다

Playback Speed는 다른 미디어 유형마다 다르게 작용합니다:

  • WAV: 컨텐츠가 보다 높거나 낮은 음조로 재생됩니다.
  • MIDI: 컨텐츠가 보다 빠르거나 느린 템포로 재생됩니다.

참고: 이 값은 해당 상위 오브젝트의 Playback Speed로 곱해져서 실제 재생 속도를 결정합니다.

참고: Interactive Music 재생 인스턴스 동안 활성화된 세그먼트의 실제 재생 속도는, 같은 재생 인스턴스에서 재생 중인 모든 세그먼트에 적용됩니다. 이는 두 세그먼트 간의 전환 시, 크로스페이드가 적용될 때 특히 중요합니다. 페이드인(새로 활성화된) 세그먼트의 재생 속도가 페이드아웃(이전 활성화된) 세그먼트에 적용됩니다.

Default value: 1
Range: 0.25 to 4
