Bypass All

오브젝트에 삽입된 모든 Effect가 처리될지 여부를 정합니다. 체크 상자를 선택해서 모든 Effect를 제거합니다. 체크 상자를 해제해서 Effect를 복원합니다.

이 옵션은 다음과 같은 상황에서 유용합니다:

  • 게임을 프로파일링할 때 이 옵션을 켜고 끄면서 해당 Effect가 사운드 디자인 및 CPU 사용량에 미치는 영향을 Performance Monitor에서 확인할 수 있습니다.
  • 플랫폼별로 사운드 디자인을 사용자 정의할 때 오브젝트의 Effect 연결을 해제한 후, 처리 성능이 낮은 플랫폼에서는 모든 Effect를 바이패스해서 사용할 수 있습니다.

또한, RTPC에서 Bypass 옵션을 사용하거나 이벤트에서 Bypass Effect Actions를 사용하여 특정 게임 상황이나 Event에 대해 Effect를 제거할 수 있습니다.

Default value: false
