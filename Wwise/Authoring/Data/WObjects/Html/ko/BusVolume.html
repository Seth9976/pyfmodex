Bus Volume

버스 또는 Auxiliary Bus 레벨에서 오디오 신호에 적용되는 감쇠 (레벨 또는 진폭)를 말합니다. 더 많은 정보를 위해서는 보이스 파이프라인 이해하기 를 참조하세요.

참고: 슬라이더의 기본 범위는 -96에서 +12까지입니다. 이 한계를 넘어가려면 값을 직접 입력하거나, 입력란에 포커스가 있는 상태에서 마우스를 드래그하면 됩니다. Bus Volume 설정에는 기본적으로 Vertical Fader가 표시됩니다. Horizontal Slider로 전환하려면 Volume 설정을 우클릭하고 Use Horizontal Slider for Volume을 선택하세요.

Default value: 0
Range: -200 to 200
Units: dB
