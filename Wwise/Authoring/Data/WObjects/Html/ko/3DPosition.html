3D Position

3D 위치 지정 (감쇠 및 공간화)을 위해 위치를 계산할 방법을 정의합니다.

  • Emitter(방사체): 위치 지정이 게임에 의해 정의됩니다.
  • Emitter + Automation(방사체 + 오토메이션(자동화)): 방사체 게임 오브젝트 주변으로 User-Defined(사용자가 정의한) 위치 지정 정보가 자동화됩니다. Automation 버튼을 통해 편집됩니다.

참고: Emitter + Automation을 사용하는 사운드에서는 Wwise의 공간 오디오 기능이 제한됩니다. Diffraction과 Transmission 처리 기능은 Emitter + Automation을 사용하는 사운드에서는 비활성화됩니다. 또한, 오토메이션 오프셋이 초기 반사음 처리에 적용되지 않습니다. 반사음은 기본 Game Object 위치를 기준으로 계산됩니다.

  • Listener + Automation: 청자 게임 오브젝트 주변의 User-Defined 위치 오토메이션. Automation 버튼을 통해 편집됩니다.

참고: Listener + Automation을 사용하는 사운드에서는 Wwise의 공간 오디오 기능이 제한됩니다. 여기에는 초기 반사음 처리, Distance Probe, 회절 및 전달, 그리고 Room 센드가 포함됩니다.

Default value: Emitter
