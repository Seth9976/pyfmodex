Height Spread

Height Spread는 평면 구성에서 패닝이 발생할 경우 높이를 기반으로 최소한의 확산을 자동으로 계산하거나, 7.1.4와 같은 반구형 구성에서 팬닝이 발생할 경우 오직 음성 높이(negative elevation)을 기준으로 계산됩니다. 이는 3D 사운드가 리스너 위나 아래로 지나갈 때 패닝을 부드럽게 처리하는 데 도움을 줍니다.

Height Spread는 기본적으로 항상 활성화되어 있으며, 사운드에 감쇠가 없더라도 활성화됩니다. 이 속성을 선택 해제해야만 Height Spread을 비활성화할 수 있습니다. 사운드에 감쇠를 사용하지 않으면서 Height Spread를 비활성화하고 싶다면 이를 위한 특정 Output Bus Volume을 가진 Attenuation ShareSet을 만들어야 합니다.

더 자세한 정보는 Height Spread의 효과 를 참고하세요.

Default value: true
