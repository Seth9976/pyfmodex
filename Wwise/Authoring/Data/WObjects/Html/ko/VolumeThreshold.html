Volume Threshold

기본 음량 레벨을 말하며, 이 음량 밑으로 떨어질 때 Property Editor의 Advanced Settings 탭에서 정의한 작동 방식이 보이스를 관리하게 됩니다. 예를 들어 음량 한계값 미만의 보이스는 다음의 동작을 실행할 수 있습니다:

  • Continue to play
  • Be killed
  • Sent to the virtual voice list

이 값은 Wwise SDK API를 사용해서 덮어 쓸 수 있습니다.

참고: Even if it continues to play or becomes virtual, the voice will be inaudible while it’s still below the volume threshold. 참고: 기본 슬라이더 범위는 -96부터 0까지입니다. 이 제한 범위를 초과하시기 위해서는 값을 직접 입력하시거나 마우스가 편집 조종에 초점이 맞춰져 있을 때에 마우스 휠을 굴리시면 됩니다.

For more information, see Specifying volume thresholds for a project.

Default value: -80
Range: -96 to 0
