AK::SoundEngine::AddOutput: Output already exists, not added a second time. Shareset: ... Device: ...

“AK::SoundEngine::AddOutput: 输出已经存在，不能再次添加。共享集: …设备:…”游戏代码调用了 AK::SoundEngine::AddOutput 函数并尝试添加系统中已经存在的音频输出。此项并非严重错误。现有输出会继续正常工作。

推荐的解决步骤：

  • 移除函数调用。

  • 重新估算参数 in_settings.audioDeviceShareset 和 in_settings.idDevice 以指向不同的物理设备。
