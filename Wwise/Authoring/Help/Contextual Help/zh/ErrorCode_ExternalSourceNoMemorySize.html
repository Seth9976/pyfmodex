No memory size provided in AkExternalSource structure, but data pointer was used. Fill uiMemorySize.

“AkExternalSource 结构中未提供内存大小，但使用了数据指针。填写 uiMemorySize。”在结合 External Source（外部源）使用 AK::SoundEngine::PostEvent 并在内存中预先加载了数据时，必须为 AkExternalSourceInfo 结构提供有效的 pInMemory 和 uiMemorySize。

推荐的解决步骤：

  • 使用预加载 WEM 文件的全部字节数填写 uiMemorySize。

  • 改为在 szFile 中指定文件名称，以便使用从磁盘流播放的版本。
