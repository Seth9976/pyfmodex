sceAudioOut2ContextCreate Failed.

“sceAudioOut2ContextCreate 失败。”在初始化 PlayStation 的核心音频输出系统时出现严重错误。通常会返回错误代码 -2144960510 或 SCE_AUDIO_OUT2_ERROR_OUT_OF_RESOURCE (0x80268002)。若 Wwise 尝试在应用程序中的另一系统或声音引擎使用某些 Object Port（对象端口）创建 sceAudioOut2 Context 后使用 3D Audio（3D 音频）所需的一个或多个 Object Port 初始化 sceAudioOut2 Context，则可能会出现这种错误：PS5 仅支持使用 Object Port 初始化一个 sceAudioOut2 Context。为了避免发生这一错误，可选择不使用 Object Port 来初始化 Wwise，或者禁用其他声音引擎对 3D Audio 的支持。若返回其他错误代码或在不使用 Object Port 初始化 Wwise 的情况下依然发生此错误，请联系 Audiokinetic 技术支持部门。
