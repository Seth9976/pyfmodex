Portals cannot connect rooms that are in the same Reverb Zone hierarchy.

Portalが接続する各ルームは、別々のRoom階層に属する必要があります。Room階層を AK::SpatialAudio::SetReverbZoneで作成すると、ルーム間で親子関係が設定されます。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。

  • AkPortalParams::FrontRoomとAkPortalParams::BackRoomで指定し、 AK::SpatialAudio::SetPortalに渡した各Roomが、同じ親を共有せず、Reverb Zoneによって作成された同一のRoom階層に属していないことを確認します。

  • AK::SpatialAudio::SetReverbZoneは、すでにポータルによって接続されたルーム間に親子関係や兄弟関係を設定することはできません。
