sceAudioOut2ContextCreate Failed.

PlayStation의 코어 오디오 출력 시스템 초기화 도중 치명적인 오류가 발생했습니다. 보통 오류 코드 -2144960510이나 SCE_AUDIO_OUT2_ERROR_OUT_OF_RESOURCE (0x80268002)가 반환됩니다. 이러한 오류 종류는 애플리케이션에 있는 또다른 시스템이나 사운드 엔진이 이미 몇몇의 Object Port로 sceAudioOut2 Context를 생성한 후에 Wwise가 한 개 이상의 Object Port로 (3D Audio를 위해 필수) sceAudioOut2 Context를 초기화하려고 할 경우 발생할 수 있습니다. PS5는 Object Port에 한 개의 sceAudioOut2 Context 초기화만 지원합니다. 이 오류를 피하려면 Object Port가 없는 채로 Wwise를 초기화하거나 다른 사운드 엔진의 3D Audio 지원을 비활성화하세요. Object Port 없이 Wwise를 초기화하더라도 다른 오류 코드가 리턴되거나 이 오류가 계속해서 발생하면 Audiokinetic 지원팀에 연락해 주세요.
