Unsupported feedback connection to Audio Objects bus.

보이스 그래프에서 감지된 활성화된 피드백 연결의 도착지가 Audio Objects 버스입니다. Audio Objects 버스로의 피드백 연결은 지원되지 않습니다. 그렇기 때문에 해당 버스 구성을 특정 스피커 구성이나 'Same as Main Mix'처럼 해당 입력 연결의 다운믹스를 실행하는 버스 구성으로 변경해 주세요. 혹은 피드백 연결이 복잡한 오디오 전파나 공간 음향 구성으로 인한 결과가 아닐 경우 피드백 연결을 생성한 상황을 피하는 방법을 시도해볼 수 있습니다. 피드백 연결은 보통 특정 이미터-리스너 관계가 보이스 그래프에서 주기를 형성하면서 발생합니다.
