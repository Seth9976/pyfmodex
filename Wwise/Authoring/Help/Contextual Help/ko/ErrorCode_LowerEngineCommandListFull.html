권장 해결 단계:

  • Increase the memory limit by changing AK::MemoryArena::AkMemoryArenaSettings::uMemReservedLimit before calling AK::SoundEngine::Init. 초기화 설정에 대한 더 많은 정보는 고급 사운드 엔진 통합을 봐주세요.

  • Playback Limits와 Virtual Voices를 사용하여 보이스를 제거해서 메모리를 더 잘 관리해 주세요.
