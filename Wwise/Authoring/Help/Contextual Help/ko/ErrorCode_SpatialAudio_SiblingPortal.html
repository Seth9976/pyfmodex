Portals cannot connect rooms that are in the same Reverb Zone hierarchy.

Portal은 반드시 별도의 Room 계층 구조에 속하는 공간으로 연결되어야 합니다. 공간 계층 구조는 AK::SpatialAudio::SetReverbZone으로 생성되며, 공간 간의 상하위 관계를 형성합니다.

권장 해결 단계:

  • Profiler Settings(Alt-G)에서 API Calls을 활성화하여 게임이 수행하는 API 호출을 프로파일링하세요. 상황을 다시 시험하여 이 오류가 일어나기 전 해당 함수에 제공된 매개 변수 값을 확인하세요.

  • AkPortalParams::FrontRoom과 AkPortalParams::BackRoom으로 지정되었으며 AK::SpatialAudio::SetPortal 로 전달되는 Room이 동일한 상위 계층을 공유하지 않으며 Reverb Zone에 의해 생성된 Room과 동일한 계층 구조에 있지 않은지 확인하세요.

  • AK::SpatialAudio::SetReverbZone은 이미 포털과 연결된 공간과 상하위 혹은 자매 관계를 형성해서는 안됩니다.
